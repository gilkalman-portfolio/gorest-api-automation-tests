# GoRest API Automation Framework 

## Overview
This project is an automated testing suite for the GoRest SaaS API. It covers a full User Management lifecycle (CRUD) and demonstrates professional API testing methodologies.

## Technical Highlights
* **Dynamic Scripting:** Pre-request scripts in JavaScript to generate unique test data.
* **API Chaining:** Passing variables (User ID) between requests to create a seamless E2E flow.
* **Environment Management:** Use of variables for secure and flexible configuration.
* **Validation:** Assertions for status codes, data types, and business logic integrity.

## How to execute
1. Import the JSON files to Postman.
2. Set your Bearer Token in the `access_token` environment variable.
3. Run the collection via the Postman Runner.
